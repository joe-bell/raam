# raam

> (estonian) ˈrɑːm _**n**._ frame.

Forget about browser support – **use flex `gap` today**.

When using new CSS features, a `@supports` query is the safest route to take.  
This [isn't an option](https://github.com/w3c/csswg-drafts/issues/3559) for flex `gap`.

**raam** offers an alternative approach.
